# 🎬 Movie Recommendation System

## 📌 Overview

This is a **Machine Learning based Movie Recommendation System** that suggests similar movies based on user input.
The project is built using **Flask (Backend)** and **React (Frontend)**.

---

## 🚀 Features

* 🔍 Search any movie
* 🎯 Get similar movie recommendations
* ⚡ Fast and simple UI
* 🧠 ML-based recommendation (content-based filtering)
* 🌐 Full-stack project (Frontend + Backend)

---

## 🛠️ Tech Stack

### 🔹 Frontend

* React.js
* HTML, CSS, JavaScript

### 🔹 Backend

* Python
* Flask

### 🔹 Machine Learning

* Pandas
* Scikit-learn
* Cosine Similarity

## ⚙️ How to Run the Project

### 🔹 1. Clone Repository

```
git clone https://github.com/YOUR_USERNAME/movie-recommender.git
cd movie-recommender
```

---

### 🔹 2. Backend Setup

```
cd backend
python -m venv venv
venv\Scripts\activate
pip install flask pandas scikit-learn flask-cors
python app.py
```

---

### 🔹 3. Frontend Setup

```
cd frontend
npm install
npm start
```
